# RSA: Serijska i Paralelna Implementacija

Ovaj projekat demonstrira i vizualizira performanse serijske i paralelne implementacije RSA algoritma. C++ aplikacija vrši enkripciju i dekripciju, dok Python GUI (Tkinter) služi za interakciju, pokretanje procesa i vizualizaciju rezultata.

## Ključne karakteristike 

-   **RSA Implementacija**: Efikasan C++ kod za enkripciju i dekripciju.
-   **Paralelizacija**: Korištenje **OpenMP** biblioteke za ubrzanje procesa dekripcije.
-   **Mjerenje performansi**: Precizno mjerenje i usporedba vremena izvršavanja serijske i paralelne verzije.
-   **Grafički interfejs (GUI)**: Intuitivni interfejs napravljen u Pythonu (Tkinter) za jednostavno korištenje.
-   **Vizualizacija podataka**: Automatsko generiranje grafa (pomoću `matplotlib`) koji prikazuje postignuto ubrzanje (*speedup*).
